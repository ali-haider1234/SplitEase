# SplitEase

SplitEase is a simple shared expense tracker for group outings. Add participants, log expenses, and see who owes whom at the end.

## Features
- Add participants
- Record expenses with payers and split options
- View balances and simplified settlements
- Personal spending summary
- Export a PDF report
- Reset all data

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap 5

## Getting Started
Open `index.html` in your browser.

## Deployment (Vercel)
1. Import this repo in Vercel
2. Framework Preset: Other
3. Build Command: leave empty
4. Output Directory: leave empty
5. Deploy
