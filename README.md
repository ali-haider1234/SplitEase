# SplitEase

SplitEase is a simple shared expense tracker for group outings. Add participants, log expenses, and see who owes whom at the end.

## Features
- Add participants
- Record expenses with payers and split options
- View balances and simplified settlements
- Personal spending summary
- Export a PDF report
- Reset all data

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap 5

## Getting Started
Open `https://split-ease-opal.vercel.app/?` in your browser.
